There are many Unix commands ported to many platforms made just for educational
purposes.

## Installation

```
cargo install --git https://github.com/TeoDev1611/tux --locked
```

## Avaliable commands:

- `which`
- `touch`
- `ls`
- `cat`
- `rm`
